# To-Do List

Browser-based task list: add items, mark them complete, remove them, and switch between All, Active, and Done views. Tasks are stored in `localStorage`, so they persist after a page refresh.

## Live demo

Live demo link is included in the email you send to the recruiter.

## Run locally

Open `index.html` in a current browser (Chrome, Firefox, Edge, or Safari). No install or build step.

## Implementation

Single HTML file: markup, styles, and scripts together. Plain JavaScript, no dependencies.
